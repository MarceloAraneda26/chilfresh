# Reporte Interactivo Chilfresh

Proyecto React/Vite generado a partir del HTML del reporte Chilfresh.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Estructura

```text
src/
  assets/              Fondo visual corporativo-tech
  components/          Secciones y componentes de UI
  data/                Datos del reporte, hitos, modulos, riesgos y minutas
  hooks/               Hooks de comportamiento visual
  App.jsx              Composicion principal
  styles.css           UI/UX dark corporate-tech
```

## Funcionalidad

- Linea de tiempo interactiva con filtros y busqueda.
- Panel de detalle por hito.
- KPIs, donut y avance por modulo.
- Buscador y detalle por modulo.
- Estimacion por escenario.
- Riesgos y matriz de concentracion.
- Resumen consolidado y por minuta.
- UI responsive con soporte `prefers-reduced-motion`.
